## Installation for Termux

```sh
$ pkg update
$ pkg upgrade
$ pkg install python3
$ pkg install git
$ git clone https://github.com/alinxmutz/simplesb
$ cd simplesb
$ cd helloworld
$ python3 -m pip install -r requirements.txt
$ python3 helloworld.py
```

## Installation for VPS

```sh
$ git clone https://github.com/alinxmutz/HelloWorld
$ cd HelloWorld
$ python3 -m pip install -r requirements.txt
$ python3 helloworld.py
```
